# 📈 Predictive Sales Forecast Dashboard

Predictive Sales Forecast Dashboard is a Python-based Data Analytics and Machine Learning project developed as part of my Data Analytics Internship at CodeTech IT Solutions.

The project analyzes historical sales data, predicts future sales trends, and provides business insights through an interactive dashboard. It helps transform raw sales data into meaningful forecasts for better decision-making.

---

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Scikit-Learn
* Streamlit
* Jupyter Notebook

---

## ✨ Key Features

* Sales Trend Analysis
* Region-wise Sales Analysis
* Category Contribution Analysis
* Top Products Analysis
* AI-Based Sales Forecasting
* 6-Month Sales Prediction
* Business Insights Generation
* Interactive Dashboard
* Download Filtered Data
