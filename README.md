# 📈 Predictive Sales Forecast Dashboard

Predictive Sales Forecast Dashboard is a Python-based Data Analytics and Machine Learning project developed as part of my Data Analytics Internship at CodeTech IT Solutions.

The project analyzes historical sales data to identify trends, forecast future sales performance, and generate meaningful business insights through an interactive dashboard. It combines data analytics, visualization, and predictive forecasting techniques to support data-driven decision-making.

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Prophet
* Streamlit
* Jupyter Notebook
* GitHub

## ✨ Key Features

* Data Cleaning and Preprocessing
* KPI Analysis
* Historical Sales Trend Analysis
* Region-wise Sales Analysis
* Category-wise Sales Analysis
* Top Products Analysis
* AI-Based Sales Forecasting
* 6-Month Sales Prediction
* Interactive Dashboard
* Business Insights Generation
* Download Filtered Data Functionality

## 📈 Project Workflow

Sales Data → Data Cleaning → Data Analysis → Forecasting → Dashboard Development → Business Insights → Decision Making
